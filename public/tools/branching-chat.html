How this works

A branching-conversation prototype for whatupwolf.com/lab.

A conversation is usually a straight line. Here it is a tree. Every message — yours and Claude's — is a node on the canvas, linked to the one it followed.

  • Send a message and it attaches under the currently selected node; Claude's reply attaches under that. The path from the top of the tree down to the node you send from is the context the model sees.
  • Branch from any message (the ⑃ button, or click a node to select it) and your next message starts a new thread from that point — the original thread is left untouched, so you can explore two answers side by side.
  • Pan the canvas by dragging the background; zoom with the wheel or the +/− controls. Drag a node's bar to move it.
  • Toggle canvas ⇄ linear in the toolbar to read the selected thread as an ordinary top-to-bottom chat.
  • Anthropic-only, bring-your-own-key. The key stays in your browser and is sent only to api.anthropic.com.
